Describe the beam

Type a plain-English description — span, loading, support conditions. The AI interprets it and builds the structural model.

Switch to Design mode (optional)

Toggle + Design below the input to enable RC design. Expand Advanced settings to set concrete grade, cover, and bar preferences. Defaults (C30 · fy 500 · 40mm cover) apply if you skip this step.

3

Run the analysis

Click Run analysis (or Design beam in Design mode). The system builds the structural model, then computes SFD, BMD and deflection diagrams.

4

Review design results (Design mode only)

RC design checks cover flexure, shear and deflection to BS8110. Enter beam width b and depth h to see bar selections and link spacing.
