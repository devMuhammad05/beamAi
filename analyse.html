Use the chips row below the textarea to set code, concrete grade, cover and bar sizes. Or override inline — "use fck 35 for this one".

Run the analysis

Click Run analysis. The system builds the structural model (§01), then computes SFD, BMD and deflection diagrams (§03).

4

Review design results

§04 shows RC design checks — flexural, shear and deflection to BS8110 or EC2. Enter beam width b and depth h to see bar selections and link spacing.

Export

Download a PDF calculation sheet from §04 or a full numerical report from §05. Both are generated client-side — nothing is sent to a server.
